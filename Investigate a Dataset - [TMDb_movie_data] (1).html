Introduction¶

I will be analyzing the TMDB dataset in this project. The dataset contains information about movies collected from the TMDB. Each row contains a number of characteristics about each film. The dataset contains some of the following colomns

revenue - The amount generated from a movie

vote_avreage - Average rating of a movie by individuals

def top_10(col_name,size=10):
    #sort the given column and select the top 10
    df_sorted = pd.DataFrame(df[col_name].sort_values(ascending=False))[:size]
    df_sorted['director'] = df['director']
    plt.figure(figsize=(12,6))
    #Calculate the avarage
    #avg = np.mean(df[col_name])   
    sns.barplot(x=col_name, y='director', data=df_sorted, label=col_name)
    #plt.axvline(avg, color='k', linestyle='--', label='mean')
    if (col_name == 'profit' or col_name == 'budget' or col_name == 'revenue'):
        plt.xlabel(col_name.capitalize() + ' (U.S Dolar)')
    else:
        plt.xlabel(col_name.capitalize())
    plt.ylabel('')
    plt.title('Top 10 Directors according to Profit made')
    plt.legend()
